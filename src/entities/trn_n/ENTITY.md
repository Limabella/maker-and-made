# Trainer Bot (TRN-N)
The Trainer Bot is a vision-based posture correction agent that analyzes the user's exercise posture and provides natural language coaching.

## Key Features
- Real-time posture estimation and analysis
- Exercise repetition counting
- Posture error detection and feedback
- Personalized coaching message generation

## Architecture
- Motion Analyzer: Joint angles, phase classification
- LLM Coach: GPT-style coaching
- State Tracker: Session management

## Tech Stack
- Python
- OpenAI API (LLM coaching)
- OpenCV (video processing)
